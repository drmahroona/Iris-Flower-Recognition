# 🌸 Iris Flower Classification

A machine learning project to classify iris flowers into three species using K-Nearest Neighbors algorithm.

## 🐢 About
This program uses the famous Iris dataset to train a KNN model that can identify iris species based on sepal and petal measurements.

## 📁 Files
- `iris_classifier.py` - Main program
- `iris.csv` - Dataset
- `mahroona_model.knn` - Trained model
- `mahroona_scaler.bin` - Feature scaler
- `mahroona_encoder.bin` - Species encoder

## 🚀 How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Place `iris.csv` in the project folder
3. Run: `python iris_classifier.py`

## 📊 Features
- Automatic model training on first run
- Model saving for faster future use
- Interactive prediction mode
- Probability breakdown for predictions
